# Market Regime Intelligence Agent v1

Static demo for market regime detection and strategy review.

## What it shows

- Current Market Regime
- Recommended Strategies
- Avoid Now Strategies
- Regime Transition Monitor
- Backtest Summary
- Investment Committee View

## Run locally

Open `Market_Regime_Intelligence_Agent_v1/index.html` in a browser.

The page tries to load `sample-regime-data.json` first. If the page is opened directly from the filesystem, it falls back to inline sample data embedded in the HTML so the demo still renders.

## Project intent

This folder is isolated from the legacy Option Playbook Advisor assets and is meant to become the first standalone Market Regime Intelligence Agent v1 demo.
