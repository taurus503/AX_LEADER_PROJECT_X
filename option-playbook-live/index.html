다음 단계 연결 포인트

이 화면은 실제 데이터 연결용 HTML입니다. 지금은 Yahoo Finance 공개 차트 API로 KOSPI200 시계열을 읽고, 선택한 거래일의 20일 실현변동성과 skew proxy를 계산합니다.

  • 날짜 입력값을 API 요청 파라미터로 전송
  • KOSPI200 시계열과 20일 통계값을 JSON으로 수신
  • 받은 값으로 활성 regime만 자동 강조
  • 실패 시에는 오류 메시지를 카드에 직접 표시
