Insight 01

알파와 베타를 분리해야 함

Brinson 분해로 비중 효과와 선택 효과를 나누고, 추가로 팩터 귀속을 붙여 성과의 원인을 분리합니다.

Insight 02

레짐 적합성이 중요

현재 국면이 Bull, Transition, Bear 중 어디인지 먼저 판별하고, 전략 선택이 그 상태와 맞았는지 확인합니다.

Insight 03

데이터 정합성 우선

결측, 상장폐지, 정지 종목은 배제하고, 표준화와 시계열 경계를 엄격히 지켜 해석의 오류를 줄입니다.

Insight 04

사람 검토가 마지막 단계

시스템은 진단과 초안을 돕지만, 최종 제출과 판단은 반드시 사람이 수행하도록 설계합니다.

레짐 클러스터

10개 상태를 3개 구간으로 묶어 현재 환경과 전략 선택의 적합성을 빠르게 판단합니다.

Bull / Calm

States 1-3 · Low Volatility
State 1
추세가 안정적이고 거래 환경이 우호적일 때입니다.
State 2
낮은 변동성과 완만한 상승이 이어지는 구간입니다.
State 3
강한 위험선호가 나타나는 마지막 Calm 구간입니다.

Transition / Mixed

States 4-7 · Medium Volatility
State 4
방향성이 약해지고 정보 우위가 흔들리기 시작하는 구간입니다.
State 5
혼조 국면의 시작으로, 기존 가정이 깨지는지 점검해야 합니다.
State 6-7
전환이 진행되는 구간으로, 베타 축소와 리스크 관리가 우선입니다.

Bear / Crisis

States 8-10 · High Volatility
State 8
하락 압력이 뚜렷해지는 초기 경계 구간입니다.
State 9
변동성이 급격히 커지는 국면으로, 과거 위기와 비교가 중요합니다.
State 10
2008, 2020 수준의 크래시 국면으로 방어적 재설정이 필요합니다.

보고서와 업데이트 신호

리포트 필수 항목

  • Risk & Return Summary: CAGR, Sharpe, Sortino, MDD
  • Attribution Breakdown: 시스템적 팩터 vs 선택 알파
  • Regime Match: 현재 국면과 전략의 적합성
  • Lessons Learned: 성공/실패 원인의 요약
  • Disclosure: gross-of-fees / net-of-fees 명시
