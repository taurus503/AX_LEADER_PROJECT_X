# AGENTS.md — 민원 지식봇 작업 원칙

## Context (맥락)
○○생명 교육용 민원 지식봇. 공개·가상·더미 FAQ만 사용. 실고객·계약번호·주민번호·실약관 금지.

## Goal
질문 → 관련 FAQ 검색 → 근거(FAQ #) 포함 참고용 답변. 범위 밖·금액은 상담사 연결.

## WHETHER before HOW
꼭 필요한가 / 공개·더미로 설명되나 / 사람 승인 없이 되돌릴 수 있나.

## 확인 게이트 (비개발자용)
① 질문→관련 FAQ ② 답변에 '근거: FAQ #' ③ 범위 밖·금액→상담사 연결 ④ 화면·코드·깃에 비밀키 없음.

## Failure / HITL (사람이 책임)
정보 부족·금액·확정·발송은 단정 금지 → 상담사/약관 확인, 사람 승인. "배포됨 ≠ 동작됨".
