# TEST_3 - Regime Detection Engine

This folder contains the final Codex 3 HTML artifact and deployment metadata for the AX Leaders group project.

## Included files

- `index.html`: final HTML version of the Codex 3 Validation Lite Agent.
- `vercel.json`: static Vercel routing configuration.
- `package.json`: deployment helper script metadata.
- `deployment-info.md`: Vercel production deployment reference.
- `.gitignore`: excludes local credentials, dependencies, and build/runtime artifacts.

## Production deployment

- Public URL: https://vercel-codex3-validation-lite.vercel.app
- Vercel project: `sunhye/vercel-codex3-validation-lite`

## Excluded from GitHub

The following local artifacts are intentionally not included:

- `.env`, API keys, tokens, and local credential files
- `.vercel/` and Vercel local auth/link state
- `node_modules/` and package manager caches
- build output folders and log files
